з¬¬4з«  жөҒзЁӢжҺ§еҲ¶

  1. иҜӯеҸҘе’ҢеӨҚеҗҲиҜӯеҸҘ

    2. JavaиҜӯиЁҖдёӯиҜӯеҸҘеҸҜд»ҘжҳҜд»ҘеҲҶеҸ·вҖң;вҖқз»“е°ҫзҡ„з®ҖеҚ•иҜӯеҸҘпјҢд№ҹеҸҜд»ҘжҳҜз”ЁдёҖеҜ№иҠұжӢ¬еҸ·вҖң{}вҖқжӢ¬иө·жқҘзҡ„еӨҚеҗҲиҜӯеҸҘгҖӮ
    3. JavaиҜӯиЁҖдёҚе…Ғи®ёеңЁдёӨдёӘеөҢеҘ—зҡ„еӨҚеҗҲиҜӯеҸҘеҶ…еЈ°жҳҺдёӨдёӘеҗҢеҗҚзҡ„еҸҳйҮҸгҖӮ
    4. Javaдёӯзҡ„жіЁйҮҠдёәзЁӢеәҸжҸҗдҫӣж–ҮжЎЈиҜҙжҳҺгҖӮеҪўејҸпјҡ
      1. еҚ•иЎҢпјҡ //
      2. еӨҡиЎҢпјҡ/* */
      3. ж–Үд»¶жіЁйҮҠ(Javaзү№жңүзҡ„)пјҡ/** */
      4. ж–Үд»¶жіЁйҮҠдә§з”ҹдёҖдёӘHTMLж–ҮжЎЈпјҢд»ҺиҖҢдёәзЁӢеәҸжҸҗдҫӣж–ҮжЎЈиҜҙжҳҺгҖӮ

  2. йЎәеәҸз»“жһ„гҖҒеҲҶж”Ҝз»“жһ„(ifеҸҢи·ҜжқЎд»¶з»“жһ„гҖҒifеҚ•и·ҜжқЎд»¶з»“жһ„гҖҒifеӨҡйҮҚжқЎд»¶йҖүжӢ©з»“жһ„(if-elseеөҢеҘ—))гҖҒswitchеӨҡи·ҜйҖүжӢ©

  3. еҫӘзҺҜз»“жһ„пјҡwhileгҖҒdo-whileгҖҒforгҖҒеӨҡйҮҚеҫӘзҺҜ(еҫӘзҺҜеөҢеҘ—)

  4. еҫӘзҺҜдёӯзҡ„и·іиҪ¬иҜӯеҸҘпјҡbreakгҖҒcontinueгҖҒreturn

    1. breakиҜӯеҸҘ:дҪҝзЁӢеәҸзҡ„жөҒзЁӢд»ҺдёҖдёӘиҜӯеҸҘеқ—(switchжҲ–еҫӘзҺҜз»“жһ„)еҶ…и·іеҮәгҖӮ
    2. continueиҜӯеҸҘ:з»ҲжӯўеҪ“еүҚиҝҷдёҖиҪ®(ж¬Ў)зҡ„еҫӘзҺҜпјҢиҝӣе…ҘдёӢдёҖиҪ®пјҲж¬ЎпјүеҫӘзҺҜгҖӮ
    3. returnиҜӯеҸҘ:
      1. з”ЁжқҘдҪҝзЁӢеәҸд»Һж–№жі•пјҲеҮҪж•°пјүдёӯиҝ”еӣһпјҢе№¶иҝ”еӣһдёҖдёӘеҖјгҖӮ
      2. return иҝ”еӣһеҖјпјӣ

ж•°з»„дёҺеӯ—з¬ҰдёІ

  1. ж•°з»„зҡ„еҹәжң¬жҰӮеҝө

    1. ж•°з»„зҡ„жҰӮеҝөпјҡзӣёеҗҢж•°жҚ®зұ»еһӢзҡ„е…ғзҙ жҢүдёҖе®ҡйЎәеәҸжҺ’еҲ—зҡ„йӣҶеҗҲгҖӮJavaдёӯпјҢж•°з»„е…ғзҙ еҸҜд»Ҙдёәз®ҖеҚ•ж•°жҚ®зұ»еһӢпјҢд№ҹеҸҜд»ҘдёәеҜ№иұЎгҖӮ

    2. JavaиҜӯиЁҖеҶ…еӯҳеҲҶй…Қй—®йўҳпјҡ

      1. ж ҲеҶ…еӯҳпјҡе®ҡд№үзҡ„еҹәжң¬зұ»еһӢзҡ„еҸҳйҮҸе’ҢеҜ№иұЎзҡ„еј•з”ЁеҸҳйҮҸпјҢи¶…еҮәдҪңз”Ёеҹҹе°ҶиҮӘеҠЁйҮҠж”ҫгҖӮ
      2. е ҶеҶ…еӯҳпјҡеӯҳж”ҫз”ұnewиҝҗз®—з¬ҰеҲӣе»әзҡ„еҜ№иұЎе’Ңж•°з»„гҖӮз”ұJavaиҷҡжӢҹжңәзҡ„иҮӘеҠЁеһғеңҫеӣһж”¶еҷЁжқҘз®ЎзҗҶгҖӮ
      3. еј•з”ЁеҸҳйҮҸпјҡдёәж•°з»„жҲ–еҜ№иұЎиө·зҡ„дёҖдёӘеҗҚз§°гҖӮеңЁе ҶдёӯеҲӣе»әдёҖдёӘж•°з»„жҲ–еҜ№иұЎеҗҺпјҢиҝҳеҸҜеңЁж Ҳдёӯе®ҡд№үдёҖдёӘеј•з”ЁеҸҳйҮҸеӯҳж”ҫиҜҘж•°з»„жҲ–еҜ№иұЎеңЁе ҶеҶ…еӯҳдёӯзҡ„йҰ–ең°еқҖпјҲеҜ№иұЎзҡ„еҸҘжҹ„пјүпјҢд»ҘеҗҺе°ұеҸҜеңЁзЁӢеәҸдёӯдҪҝз”Ёж Ҳзҡ„еј•з”ЁеҸҳйҮҸжқҘи®ҝй—®е Ҷдёӯзҡ„ж•°з»„жҲ–еҜ№иұЎгҖӮ
      4. ж•°з»„е’ҢеҜ№иұЎеңЁжІЎжңүеј•з”ЁеҸҳйҮҸжҢҮеҗ‘е®ғж—¶пјҢе°ұдјҡеҸҳжҲҗеһғеңҫпјҢдёҚиў«дҪҝз”ЁпјҢдҪҶеҚ еҶ…еӯҳпјҢеңЁйҡҸеҗҺдёҚзЎ®е®ҡзҡ„ж—¶й—ҙйҮҠж”ҫгҖӮ

    3. ж•°з»„зҡ„дё»иҰҒзү№зӮ№

      1. ж•°з»„жҳҜзӣёеҗҢж•°жҚ®зұ»еһӢзҡ„е…ғзҙ зҡ„йӣҶеҗҲпјӣ
      2. ж•°з»„дёӯзҡ„еҗ„е…ғзҙ жҳҜжңүе…ҲеҗҺйЎәеәҸзҡ„пјҢе®ғд»¬еңЁеҶ…еӯҳдёӯжҢүз…§иҝҷдёӘе…ҲеҗҺйЎәеәҸиҝһз»ӯеӯҳж”ҫеңЁдёҖиө·пјӣ
